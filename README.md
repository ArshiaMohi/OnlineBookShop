# 📚 Online Book Shop

A backend application for managing an online book store, developed using Spring Boot and MySQL.

## 🚀 Features

- Book Management
  - Add books
  - Update books
  - Delete books
  - View books

- Category Management

- Database Integration using MySQL

- RESTful API Architecture

- Layered Architecture
  - Controller Layer
  - Service Layer
  - Repository Layer
  - DTO Layer

## 🛠 Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- REST API

## 📂 Project Structure

src
├── controller
├── service
├── repository
├── dto
├── model
└── config

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/ArshiaMohi/OnlineBookShop.git
```

### Configure Database

Update your database settings in:

```properties
application.properties
```

### Run the project

```bash
mvn spring-boot:run
```
